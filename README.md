# svm
S-lang Virtual Machine, 一种栈式虚拟机

## Operand Code
| Name  | Code |    Description                |
|-------|------|-------------------------------|
| Nop   | 0x00 |                               |
| Push8 | 0x01 | Push 8bit data                |
| Add8  | 0x02 | Add two 8bit numbers in stack |
